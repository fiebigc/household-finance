# Cursor Rules

This project expects the agent to execute tasks directly using available tools.

## Working style

- Always run CLI commands when needed instead of only suggesting them.
- Prefer first-party tools (Read/Write/Edit, rg/Glob, MCP tools) where they fit best.
- Use MCP integrations whenever relevant to complete the task end-to-end.
- Only ask the user to run commands manually when blocked by permissions, auth, or required local interaction.
